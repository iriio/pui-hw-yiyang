About TONE

TONE started as my exploration into web audio and synthesis. I wanted to create something that makes sound design accessible and fun to experiment with, even if you're just getting started with music production.

What's Included

Sound Engine

Two oscillators with classic waveforms, a filter section, and some effects to shape your sound.

Step Sequencer

16 steps for both melodic and drum patterns. Record notes live or input them step by step.

Visual Feedback

Real-time waveform display helps you understand how different settings affect your sound.

Built With

  • Web Audio API
  • Tone.js
  • Canvas
  • Vanilla JS

Development

This was my final project for my web audio course. I learned a lot about audio processing, UI design, and real-time visualization. Feel free to check out the code or send me feedback.
